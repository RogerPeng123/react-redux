# 在React项目中使用redux

主要实现在react项目中hooks使用redux

### 下载项目

```
git clone https://github.com/RogerPeng123/react-redux.git
```

### 进入项目,安装依赖包
```
yarn
```

### 执行项目
```
yarn run start
```

## Learn More

该案例仅仅是作为在hooks种使用redux的基础使用方法,仅供学习使用参考。
如需要在正式项目中使用,一切风险与作者无关。
